# Seasonal Calendar Year

This project provides a simple program that takes a month and a day as input and outputs the corresponding season for the Northern Hemisphere. 
It includes functionality to validate both the month and day, ensuring that they are valid and within the range for each season.

## Features

- **Spring**: March 20 - June 20
- **Summer**: June 21 - September 21
- **Autumn**: September 22 - December 20
- **Winter**: December 21 - March 19

## Usage

1. The user is prompted to input the **month** and **day**.
2. Based on the input, the program will return the season for that date.
3. If the input month or day is invalid, the program will notify the user of the error.

### Example
Input:
April
11

Output:
Spring
